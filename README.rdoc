= Monokai BBEdit Color Scheme

A simple port of the Monokai TextMate theme to BBEdit.
Converted to BBEdit 10 format by Matt Sephton http://www.gingerbeardman.com

== REQUIREMENTS:

BBEdit 10.

== INSTALL:

Put Monokai.bbcolors at

    ~/Library/Application Support/BBEdit/Color Schemes/

Then activate using BBEdit Preferences: Text Colors, Color Scheme
